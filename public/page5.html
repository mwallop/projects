Web Security

There is a great deal of information about a user's computer that a web app can access. For instance:

the geolocation property- the ability to find out your location

appName and appVersion- shows the name and version of the web browser accessing the webpage

onLine- shows if the browser has an online connection

platform- shows which operating system the client is using

Below are a few resources to help give you more information about web security

Beyond security: gives a basic look at web security and what pages and places on the internet where your information is most at risk.

Web Security Basics

Martin Fowler- provides a more in depth look at web security. Also provides web designers with information and examples on how to provide better web security.

The Basics of Web Application Security

Tech Republic- shows a sort of checklist for web security that applies for web developers and users.
